# MachineLearningProject
A Full-Stack Weather Forecast Web Application built using Python, Django, and OpenWeatherAPI that provides real-time weather information in a clean and user-friendly interface. It fetches and displays real-time weather data for any city entered by the user.
Project Objectives:-
Fetch real-time weather data using API
Display city-wise temperature and weather conditions
Show humidity, wind speed, and visibility
Provide a clean and responsive user interface
