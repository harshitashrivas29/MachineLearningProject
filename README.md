About the Project

This is a weather forecast web application built using Python and Django.
The application allows users to search for any city and view real-time weather information such as temperature, humidity, wind speed, and visibility using the OpenWeather API.

This project was created to understand how backend logic works, how APIs are integrated, and how dynamic data is displayed on a web interface.

Features:-
Search weather details by city name
Fetches live weather data using OpenWeather API
Displays temperature, humidity, wind speed, and visibility
Simple and user-friendly interface

How to Run the Project:-

Clone the repository:
git clone https://github.com/harshitashrivas29/MachineLearningProject.git

Go to the project directory:
cd MachineLearningProject

Install required dependencies:
pip install django requests

Run the Django development server:
python manage.py runserver

Open your browser and visit:
http://127.0.0.1:8000/


API Setup:-
To run this project, you need an OpenWeather API key.
Create an account on OpenWeather and add your API key to the project.
